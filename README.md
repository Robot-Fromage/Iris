# Iris
Iris C++ desktop application, explore chromaticity & ICC profiles

## Description
        - Cross-Platform C++17
        - Chromaticity
        - Application
        
## Software Requirements Specification ( SRS )
        - 64bit OS - Linux, Windows or MacOS
        - CMake ( 3.11.4 or greater )
        - Git ( 2.17.0 or greater )
        - C++ compiler

## Supported Compilers
        - MinGW 64 7.xx
        - Clang LLVM xx
        - MSVC
        - GNU GCC
        
## Dependencies
        - Qt
        - Rivet
        - ULIS
        - Bokuseru
